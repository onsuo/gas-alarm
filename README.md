# gas-alarm

### **이산화탄소 농도 알림 및 환기 장치**

### **Arduino Uno**

### 사용 모듈 (제품번호, 라이브러리)

* CO2 Sensor ([MG811](https://github.com/smart-tech-benin/MG811.git))
* RGB LED (KY-016)
* LCD ([I2C LCD 1602](https://github.com/fdebrabander/Arduino-LiquidCrystal-I2C-library.git))
* Motor Driver (SZH-MDBL-002)
* Fan Motor (SZH-EK061)
